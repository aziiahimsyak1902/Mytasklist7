Deskripsi App
Aplikasi manajemen tugas harian berbasis React Native dengan tema gelap, yang memungkinkan pengguna menambah, memfilter, dan mengurutkan task berdasarkan prioritas. Dibangun menggunakan Expo dan mencakup validasi input, state management dengan hooks, serta tampilan statistik progres secara real-time. 

Fitur yang diimplementasikan:
App jalan mulus + screenshot
Flexbox konsisten, StyleSheet.create dipakai penuh, UI bersih
useState dipakai tepat, data flow benar, conditional render variatif
TextInput + KeyboardAvoidingView + validasi lengkap + feedback UX
FlatList + keyExtractor + ListEmptyComponent + desain item card rapi
